# Smart Feedback System

## Overview
A web application to collect user feedback, perform sentiment analysis, and display results on an admin dashboard with charts.

## Features
- User feedback form (Name, Email, Message)
- Sentiment analysis using the Sentiment package
- Admin dashboard:
  - Feedback history table
  - Positive, Negative, Neutral sentiment stats
  - Pie chart visualization
- Responsive design for desktop and mobile
- MongoDB database to store feedback
